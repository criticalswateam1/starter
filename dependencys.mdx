### Install Packages

<AccordionGroup>
  <Accordion icon="wrench" title="Update">
    Please update your package list by running `sudo apt update`
  </Accordion>
  <Accordion icon="pencil" title="Install Dependencys">
    `sudo apt install -y nodejs npm nginx`, This will install NodeJS, Node
    Package Manager, NGINX Webserver
  </Accordion>
</AccordionGroup>
